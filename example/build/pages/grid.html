Grid

- Grid System.

기본

<div class="ga-grid divide">
    <div class="column" /> // column 1
    <div class="column" /> // column 2
    ...
    <div class="column" /> // column 12
</div>

균등 분할

<div class="ga-grid divide-{2|4|6|8|10}">
    <div class="column" /> // column 1
    <div class="column" /> // column 2
    ...
    <div class="column" /> // column divide number 
</div>

분할 구성

<div class="ga-grid">
    <div class="column col-{number}" /> // column number
    <div class="column col-{number}" /> // column number
    ...
    <div class="column col-{number}" /> // column number 
</div>
